# Overview -
The Personal Expense Dashboard is an interactive Power BI dashboard designed to help users monitor and analyze their personal finances. It provides a clear overview of income sources, expense distribution across various categories, payment methods, and spending trends over time. The dashboard enables users to compare total income and expenses, identify major spending areas, and make informed financial decisions through intuitive and visually appealing reports. It serves as an effective tool for budgeting, expense tracking, and improving overall financial management.
# Features -
📊 Income Source Analysis – Visualizes income from different sources such as Freelance, Salary, Part-time, and Bonus.
💸 Expense Category Breakdown – Displays expense distribution across categories like Rent, Food, Medical, Bills, Transport, Shopping, Entertainment, Education, and Others.
💳 Payment Mode Analysis – Tracks expenses made using Card, Cash, and UPI.
📈 Expense Trend Analysis – Shows daily expense patterns to identify spending trends over time.
📋 Expense Summary – Provides a summary of expenses based on different payment methods.
💰 Key Performance Indicators (KPIs) – Displays Total Income and Total Expenses for a quick financial overview.
🎨 Interactive Visualizations – Includes Treemap, Pie Charts, Line Chart, and KPI Cards for easy data interpretation.
📅 Time-Based Analysis – Enables users to monitor expenses by date and observe spending behavior over time.
⚡ User-Friendly Dashboard – Clean and intuitive layout for easy navigation and better decision-making.
📉 Financial Insights – Helps users identify spending habits, compare income with expenses, and improve budgeting.
